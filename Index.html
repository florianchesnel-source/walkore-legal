Politique de confidentialité de Walkore

Dernière mise à jour : 25 avril 2026

La présente politique de confidentialité décrit la manière dont l'application Walkore (ci-après « l'Application ») collecte, utilise et protège les données des utilisateurs (ci-après « vous »). En téléchargeant, installant et utilisant Walkore, vous acceptez les pratiques décrites ci-dessous.

1. Identité du responsable du traitement

  • Nom : Florian Chesnel (Duckaktus)
  • Adresse : 16 rue de Corbeil, 77111 Soignolles-en-Brie, France
  • Adresse de contact : florianchesnel77@gmail.com

2. Données collectées

Walkore est conçu dans le respect du principe de minimisation des données. L'Application ne collecte aucune donnée d'identification personnelle directe (nom, prénom, email, numéro de téléphone, adresse, coordonnées GPS précises).

2.1 Données générées par votre utilisation

  • Identifiant de joueur anonyme (PID) : généré aléatoirement à la première ouverture, non lié à votre identité
  • Pseudo : librement choisi par vous (peut être un pseudonyme fictif)
  • Code ami : code à 6 chiffres généré pour permettre les échanges entre joueurs
  • Code de récupération : code généré pour récupérer votre progression sur un nouvel appareil
  • Statistiques de jeu : nombre de pas marchés, créatures capturées, items collectés, missions accomplies, équipe, inventaire, badges
  • Avatar : créature représentant votre profil, choisie par vous
  • Préférences de confidentialité : option « Privacy » activable pour masquer votre profil aux autres joueurs
  • Date de dernière connexion : pour afficher l'activité récente aux amis

2.2 Données capteur

  • Nombre de pas : collecté via l'API Android ACTIVITY_RECOGNITION ou le capteur podométrique de votre appareil. Cette donnée reste sur votre appareil et n'est partagée avec nos serveurs que sous forme agrégée (compteur quotidien, total) et pseudonymisée (lié à votre PID, pas à votre identité).
  • Aucune donnée GPS, aucune position géographique n'est collectée par Walkore.

2.3 Données techniques

  • Version de l'Application (pour le support technique)
  • Système d'exploitation Android (pour le diagnostic de bugs)
  • Identifiant publicitaire (AAID) : utilisé exclusivement par notre prestataire publicitaire Google AdMob (voir section 4.2)

2.4 Données échangées entre joueurs

Lorsque vous utilisez les fonctionnalités sociales (amis, échanges, missions coopératives), les informations suivantes peuvent être visibles par d'autres joueurs ayant votre code ami :

  • Pseudo, code ami, avatar
  • Pas quotidiens (dailySteps)
  • Équipe affichée publiquement (si vous ne l'avez pas masquée)
  • Offres d'échange en cours

Vous pouvez à tout moment activer l'option « Privacy » dans les paramètres pour masquer ces données aux autres joueurs.

3. Finalités du traitement

Vos données sont utilisées exclusivement pour :

  • Faire fonctionner l'Application (sauvegarde de votre progression, comptage des pas, fonctionnalités de jeu)
  • Permettre les fonctionnalités sociales (amis, échanges, missions)
  • Synchroniser votre progression entre appareils (via votre code de récupération)
  • Détecter et prévenir la triche (logs anti-tricherie anonymisés via votre PID)
  • Diffuser des publicités vidéo récompensées (via Google AdMob, voir section 4.2)
  • Améliorer l'Application (corrections de bugs, amélioration des performances)

Aucune donnée n'est utilisée à des fins de prospection commerciale, revente à des tiers, ou profilage publicitaire personnalisé par l'éditeur.

4. Destinataires des données et tiers

4.1 Google Firebase (sauvegarde cloud)

Walkore utilise Firebase Realtime Database (Google Ireland Limited) pour stocker votre progression dans le cloud.

  • Données envoyées : votre PID anonyme, pseudo, statistiques de jeu, équipe, inventaire, missions, codes amis
  • Localisation des serveurs : Union Européenne / États-Unis
  • Politique de confidentialité de Google : https://policies.google.com/privacy

4.2 Google AdMob (publicités)

Walkore intègre Google AdMob pour diffuser des publicités vidéo récompensées (Rewarded ads). L'utilisateur regarde la publicité uniquement de son plein gré pour obtenir un avantage en jeu.

  • AdMob peut collecter votre identifiant publicitaire (AAID), votre adresse IP, des informations sur l'appareil (modèle, OS, langue), et votre localisation approximative (au niveau du pays/région).
  • Politique de confidentialité Google AdMob : https://support.google.com/admob/answer/6128543
  • Vous pouvez réinitialiser votre identifiant publicitaire ou désactiver la publicité personnalisée depuis les paramètres Android : Paramètres → Confidentialité → Publicités.

4.3 Aucun autre tiers

Walkore ne partage aucune donnée avec des courtiers de données, agences marketing, ou réseaux sociaux. L'Application n'utilise aucun SDK d'analyse comportementale (pas de Facebook SDK, pas de TikTok SDK, pas d'AppsFlyer, etc.).

5. Base légale du traitement (RGPD)

Conformément à l'article 6 du Règlement Général sur la Protection des Données (RGPD), les bases légales sont :

  • Exécution du contrat : sauvegarde de progression, fonctionnalités de jeu, synchronisation cloud (article 6.1.b)
  • Intérêt légitime : sécurité, anti-tricherie, amélioration continue (article 6.1.f)
  • Consentement : pour les publicités personnalisées (article 6.1.a) — vous pouvez retirer ce consentement à tout moment via les paramètres de l'Application ou de votre appareil.

6. Durée de conservation

  • Sauvegarde cloud (Firebase) : conservée tant que votre compte de joueur reste actif. En cas d'inactivité de plus de 24 mois, les données peuvent être supprimées automatiquement.
  • Logs anti-tricherie (tamperLog) : conservés au maximum 6 mois.
  • Données locales (sur votre appareil) : conservées tant que vous n'avez pas désinstallé l'Application.
  • Demande de suppression : vous pouvez demander la suppression immédiate de toutes vos données (voir section 8).

7. Sécurité des données

  • Les communications avec les serveurs Firebase utilisent HTTPS/TLS (chiffrement de bout en bout du transport).
  • Les règles de sécurité Firebase empêchent tout joueur d'accéder ou modifier les données d'un autre joueur sans autorisation explicite.
  • Aucune donnée bancaire n'est traitée par l'Application (les paiements éventuels passent intégralement par le Google Play Store).

  • Droit à l'effacement (« droit à l'oubli ») : demander la suppression de vos données
  • Droit à la portabilité : recevoir vos données dans un format lisible
  • Droit d'opposition : refuser certains traitements

9. Mineurs

Walkore est destinée à un public âgé de 13 ans et plus. L'Application n'est pas spécifiquement destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles d'enfants de moins de 13 ans. Si vous êtes parent ou tuteur et que vous pensez que votre enfant a fourni des données personnelles, contactez-nous : nous procéderons à leur suppression rapide.

10. Cookies et stockage local

Walkore n'utilise pas de cookies web. L'Application utilise les mécanismes de stockage local Android suivants :

  • localStorage / SharedPreferences : pour la sauvegarde locale de votre progression
  • Préférences AdMob : gérées par le SDK Google AdMob conformément à sa politique

En cas de modification substantielle, vous serez informé via une notification dans l'Application lors de votre prochaine connexion. La date de « Dernière mise à jour » en haut de ce document sera actualisée.
