Supprimer mon compte et mes données — Walkore

Walkore n'utilise pas de système d'inscription traditionnel. Aucune adresse e-mail, mot de passe ou nom d'utilisateur n'est requis. Un identifiant anonyme est généré automatiquement à l'installation pour permettre la synchronisation cloud de votre progression.

Données stockées

  • Identifiant joueur anonyme (UUID auto-généré localement)
  • Pseudo affiché (si vous en avez choisi un)
  • Progression de jeu : pas effectués, créatures collectées, missions, badges, échanges
  • Code de récupération à 12 caractères (lié à votre identifiant)

Aucune donnée personnelle directement identifiante n'est collectée (pas de nom réel, e-mail, téléphone, photo, ni position GPS stockée).

Procédure de suppression

Option 1 — Désinstallation simple (effacement local uniquement)

Désinstaller Walkore depuis votre téléphone supprime toutes les données stockées localement. Les données cloud restent par défaut, pour vous permettre de restaurer votre progression si vous réinstallez l'application.

Option 2 — Suppression complète des données cloud

Pour effacer définitivement vos données du serveur, envoyez un e-mail à :

florianchesnel@duckaktus.com

avec pour objet : Suppression compte Walkore

et précisez votre code de récupération à 12 caractères (visible dans Walkore → Profil → Sauvegarde Cloud) ou votre pseudo de joueur.

Les données seront supprimées sous 30 jours. Vous recevrez une confirmation par e-mail une fois la suppression effectuée.

Conservation des données après suppression

Aucune donnée nominative n'est conservée après suppression. Toutes les données associées à votre identifiant joueur sont effacées définitivement de Firebase Realtime Database.

Des logs anonymes d'utilisation (sans identifiant joueur, à des fins statistiques agrégées) peuvent être conservés conformément à notre politique de confidentialité.
